## Features 
### Username validation
Here is the section where the user will be inputting their username in order for the system to validate or check if the username is valid or not. 
It includes an underscore, a special character and a number / digit.
Once it includes all of that, the username is valid.


### Cellphone validation
This is where the user has to input their cellphone.
The cellphone has to consist of NO MORE than 13 digits.
The cellphone number must start or contain the South African international code (e.g +27...)


### Return login status
This is where the system returns the login status to the user.
If the user has enetered everything correctly, and everything is correctly validated or checked, the system will welcome the user and display a message to say it is great to see you again. If not, the system will also send a mesaage to the user to show them that either theur username or password is incorrect and they should try it again.

### Registration section

The system will respond in relation to the validation of the information that they have entered. If the information is valid or correct, the system will register them if not, the system will NOT register the user.
